# Audiology Knowledge Base

This is a static website for your audiology glossary.

## Files to upload to GitHub

Upload these two files to the root of your GitHub repository:

- `index.html`
- `definitions.csv`

## How to update terms

1. Edit your Excel spreadsheet.
2. Use **File > Save As** or **Export**.
3. Choose **CSV UTF-8 (Comma delimited) (.csv)**.
4. Name the file exactly: `definitions.csv`
5. Upload the new `definitions.csv` to GitHub.
6. Click **Commit changes**.

GitHub Pages should update within 1–3 minutes.

## Recommended spreadsheet columns

The site currently supports these columns:

- Keyword
- Explanation
- Category
- Related Keywords
- Related Equipment
- Full Name
- Plain English
- Sales Tip
- Featured

You do not need to fill every column. Blank cells are okay.

## Tips

- Separate related keywords with commas, like: `SRT, WRS, PTA`
- Separate related equipment with commas, like: `AudioStar Pro, TympStar Pro`
- Keep the CSV file in the same folder as `index.html`.
